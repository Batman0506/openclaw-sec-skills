## 📖 关于项目

**OpenClaw SecSkills** 是一个专门为网络安全从业人员、渗透测试工程师、红蓝对抗团队整理的 **AI Agent Skills** 集合。

本项目基于 [OpenClaw](https://openclaw.ai) 框架，将传统安全工具与 AI Agent 能力相结合，让安全测试更智能、更高效。
