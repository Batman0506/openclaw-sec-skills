## 📖 关于项目

**OpenClaw SecSkills** 是一个专门为网络安全从业人员、渗透测试工程师、红蓝对抗团队整理的 **AI Agent Skills** 集合。

本项目基于 [OpenClaw](https://openclaw.ai) 框架，将传统安全工具与 AI Agent 能力相结合，让安全测试更智能、更高效。

## 🤝 贡献指南

我们欢迎所有网络安全从业者的贡献！

### 如何贡献

1. **Fork** 本仓库
2. 创建新分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启 **Pull Request**

### 提交格式

请在 PR 中按以下格式提供信息：

```markdown
## Skill 名称

- **描述**: 一句话描述功能
- **仓库链接**: GitHub URL
- **分类**: 代码审计/渗透测试/逆向工程/...
- **标签**: #java #audit #security
```

### 行为准则

- 🔒 仅用于合法授权的安全测试
- 📚 学习和研究目的
- 🤝 尊重其他贡献者
- ⚠️ 请勿用于非法用途
